---
title: "Get Referral Links"
api: "GET /referral_links"
description:
  "This endpoint returns a user's referral links, one for each merchant the
  credentials have access to."
---

### Parameters

<ParamField body="merchant_id" type="string">
  Only return referral links for this merchant
</ParamField>

### Response

<ResponseField name="merchant_id" type="string">
  Merchant ID
</ResponseField>

<ResponseField name="link" type="string">
  The URL a user can share with others
</ResponseField>

<ResponseExample>

```json
{
  "referral_links": [
    {
      "merchant_id": "9a1f0772c9ac",
      "link": "https://signup.thanx.com/pizza_bob/Jane-23uow234243"
    }
  ]
}
```

</ResponseExample>
