---
title: "Describe Tiers Configuration"
api: "GET /tier_configurations"
description:
  "This section describes endpoints that enable a third party to fetch the
  configuration of a merchant's tiers. This endpoint describes tier
  configurations for the merchants accessible via the provided credentials."
---

### Parameters

<ParamField body="merchant_id" type="string">
  Only return tier configuration for this merchant
</ParamField>

### Response

<ResponseField name="merchant_id" type="string">
  The merchant ID
</ResponseField>
